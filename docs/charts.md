# Top Charts

Get top charts

+++ Request

HTTP

```bash
https://saavn.me/search/albums?query=rockstar
```

CURL

```bash
curl -X GET 'https://saavn.me/search/albums?query=rockstar' \
 -H 'content-type: application/json'
```
